# Maven Market Analytics Dashboard | Power BI Project

## Project Overview
An end-to-end Power BI analytics project built using the Maven Market simulated retail dataset. The goal of this project was to transform raw retail data into meaningful business insights through data cleaning, modeling, DAX calculations, and interactive dashboard design.

## Project Objectives
- Analyze sales performance, profitability, customer returns, and product trends.
- Build an interactive dashboard to support data-driven decision-making.
- Apply Power BI best practices including data modeling, DAX calculations, and visualization techniques.

## Data Preparation & Modeling
- Cleaned and transformed 7 raw datasets using Power Query.
- Designed a structured **Star and Snowflake schema data model**.
- Created relationships between fact and dimension tables to optimize reporting performance.
- Applied data validation and transformation techniques to improve data quality.

## DAX Measures Created
Developed 20+ DAX measures including:
- Year-to-Date (YTD) Revenue
- 60-Day Revenue Analysis
- Last Month KPIs
- Profit Margin %
- Return Rate %
- Sales Performance Metrics

## Dashboard Features
Built an interactive **Topline Performance Dashboard** with:
- KPI Cards for sales, revenue, profit, and returns
- Matrix analysis for product and customer insights
- Geographic sales analysis using Maps
- Product performance analysis using Treemap
- Weekly sales trend analysis
- Gauge visuals for performance tracking
- Bookmarks and navigation for enhanced storytelling

## Key Business Insights
- Portland achieved **1,000+ sales in December 1998**, showing strong seasonal performance.
- Mexico experienced an increase in top product returns from **4 to 8**, resulting in a **1.2% return rate**.
